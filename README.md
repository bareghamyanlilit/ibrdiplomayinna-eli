# ԷՊՔ — Ejmiatsin State College Full-Stack App

Քանի որ կայքը գործող է անվտանգության նկատառումներից ելնելով .env ֆայլերը բացակայում են 

### 1. Server
cd server
npm install
npm run dev

### 2. Admin Panel
cd admin
npm install
npm run dev

### 3 Client (your existing site)
cd client
npm install
npm run dev
